# Listas Encadeadas em C

Este programa demonstra o uso de listas encadeadas em C.

Na primeira parte, ele trabalha com uma lista simples de números inteiros. O programa adiciona valores no final da lista, insere um número em uma posição específica, busca um valor, inverte a ordem da lista e depois divide a lista em duas partes.

Na segunda parte, o programa cria uma playlist de músicas usando lista circular duplamente encadeada. Cada música aponta para a próxima e também para a anterior. Como a lista é circular, depois da última música ela volta para a primeira.

O código também mostra como navegar para a próxima música, voltar para a música anterior, exibir a playlist, contar quantas músicas existem e percorrer a playlist uma vez sem entrar em loop infinito.

A memória usada pelas listas é alocada com `malloc` e liberada com `free` no final do programa.
